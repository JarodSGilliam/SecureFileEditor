Ctrl + h = Help Page
Ctrl + s = Save File
Alt + s = Save As
Ctrl + f = Find (will place cursor at first instance)
Ctrl + left = Moves to previous "find" result
Ctrl + right = Moves to next "find" result
Ctrl + r = Replace
Ctrl + w = Close file
Ctrl + d = Detailed file information
Ctrl + c = Command line
[Esc] to leave this screen
